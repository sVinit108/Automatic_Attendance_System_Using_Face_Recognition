# Automatic Attendance System Using Face Recognition

A face-recognition–based attendance system that automatically marks student attendance using a **live webcam feed** for a given subject.  
The project is designed with a **modular programming architecture** for clarity, scalability, and maintainability.

## Key Features

- Automatic attendance marking using real-time face recognition  
- Optimized face-encoding technique to **reduce loading and recognition time**  
- Clean and well-structured **modular codebase**  
- Supports seamless student and attendance data management  

## Core Functionalities

The system provides the following core functions:

- **Get Attendance** – retrieve attendance records  
- **Update Attendance** – mark and update attendance automatically  
- **Get Students** – fetch student details from the database  
- **Update Students** – add or update student records
